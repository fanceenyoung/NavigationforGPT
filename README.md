# NavigationforGPT
在 ChatGPT 网页版对话页右侧，显示当前会话中的用户提问的导航条。这个功能之前GPT默认就有，后来不知道为什么下掉了，vibe coding手搓了一个自己用，也拿出来给大家。目前只支持ChatGPT。

更多支持，请联系邮箱 markone_support@163.com。

In the ChatGPT web interface, this extension displays a navigation sidebar on the right side of the conversation, listing all user prompts from the current session for quick navigation.
This feature used to be available by default in ChatGPT but was later removed for unknown reasons. I rebuilt it myself using vibe coding and decided to share it.
Currently, it only supports ChatGPT.

For more information, Please contact markone_support@163.com.
